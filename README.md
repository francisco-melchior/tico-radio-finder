# Tico Radio Finder

O **Tico Radio Finder** é um aplicativo desktop leve, desenvolvido em Python e PyQt6, feito para buscar e reproduzir suas rádios online favoritas diretamente do seu sistema Linux.

## 🚀 Funcionalidades
- **Busca Rápida:** Pesquise milhares de estações de rádio em tempo real.
- **Favoritos:** Salve suas estações preferidas e acesse-as com um clique.
- **Interface Limpa:** Design minimalista em azul petróleo, focado na usabilidade.
- **Leve:** Baixo consumo de recursos, utilizando a robustez do VLC media player.

## 📥 Instalação (Debian/Ubuntu)
A forma mais fácil de instalar é através do nosso pacote `.deb`:

1. Vá até a aba **[Releases](https://github.com/francisco-melchior/tico-radio-finder/releases)** aqui no repositório.
2. Baixe o arquivo `.deb` da versão mais recente.
3. Abra o seu terminal, navegue até a pasta onde salvou o arquivo e execute:

```bash
sudo apt install ./tico-radio-finder_*.deb
```
Requisitos de Sistema

O aplicativo utiliza o VLC como motor de áudio. Certifique-se de tê-lo instalado:


```Bash
sudo apt update
sudo apt install vlc
```
